# DJIA-Stock-Sentiment-Analysis-using-News-Headlines

This project analyzes how news headlines impact the DJIA (Dow Jones Industrial Average) by applying Natural Language Processing (NLP) and Machine Learning techniques. It extracts sentiment from financial news data and evaluates its correlation with historical DJIA movements.

Features

Sentiment analysis on financial news headlines using NLTK/TextBlob
Text preprocessing: tokenization, stop-word removal, lemmatization
Feature extraction using TF-IDF vectorization
ML model training with Logistic Regression and Random Forest
Correlation analysis between sentiment scores and DJIA index trends
Data visualizations using Matplotlib/Seaborn
